# 适配器

一、介绍
适配器模式通过把一个类的接口变换成客户端所期待的另一种接口，可以帮我们解决不兼容的问题。

二、演示

```js
// 历史代码
// $.ajax({})

//自己封装的ajax,现在使用的格式
ajax({
    type: "method",
    url: "url",
    data: "data",
    dataType: "dataType"
}).done(function () { })

//做一层适配器

let $ = {
    ajax: function (options) {
        return ajax(options)
    }
}
```

三、场景

1. 封装旧接口
2. vue computed

四、总结

1. 将旧接口和使用者进行分离
2. 符合开放封闭原则
